Red Hat OpenShift adalah platform pengembangan aplikasi container open source multifaset dari Red Hat Inc. untuk pengembangan, penerapan, dan pengelolaan aplikasi


Diperkenalkan pada tahun 2011, OpenShift Online ditujukan untuk pengembang individu atau tim yang mengakses OpenShift sebagai layanan cloud publik. OpenShift Online diimplementasikan sebagai model konsumsi berdasarkan permintaan yang dihosting di platform cloud publik, termasuk Amazon Web Services ( AWS ), Microsoft Azure , dan Google Cloud .
